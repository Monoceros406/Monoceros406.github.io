生成.exe方法

准备manifest.mf
Manifest-Version:1.0

安装exe4j

  1. Project type选”JAR in EXE” mode
  2. Application info 文件名、.exe导出位置
  3. Executable info
    选 GUI程序、控制台晨雾、web服务
    exe文件名、图标
    是否允许多开
    32-bit or 64-bit中勾上
    Manifest options中DPI选Always
  4. Java invocation右侧+添加Archive选择.jar包
    右下角…添加主类
  5. JRE 最低最高 最低选1.7
    表中3个删掉，添加自己的jre
  6. Splash screen启动画面
  7. Messages默认英语 没汉语
